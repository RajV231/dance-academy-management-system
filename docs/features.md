# Features

## Authentication
Secure login system to allow only authorized administrators.

## Student Management
Administrators can:
- Add new students
- Update student details
- Maintain enrollment records

## Receipt Generation
System generates payment receipts for students and maintains financial records.

## Dashboard
Provides a centralized view for academy operations and management.

## Progressive Web App
The application includes PWA support allowing:
- Installable mobile experience
- Offline functionality
- Faster loading using cached assets
